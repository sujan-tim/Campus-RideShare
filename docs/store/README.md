# Store Release Kit

This folder contains draft metadata and launch checklists for publishing RUride to:

- Apple App Store
- Google Play

Files:

- `ios-app-store-metadata.md`
- `google-play-metadata.md`
- `privacy-policy-checklist.md`
- `screenshots-checklist.md`
- `release-commands.md`

Before submitting:

1. Replace all placeholder URLs, emails, phone numbers, and team IDs.
2. Re-check every privacy and data safety declaration against the final production app and all SDKs.
3. Capture final screenshots from release builds, not from the dev server.
4. Make sure the live backend, payments, and support contact paths are working during review.
